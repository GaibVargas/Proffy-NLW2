<h2>Rodar o projeto</h2>
<ol>
  <li>Abra a pasta do projeto, e entre na pasta web, e então execute:
  
  ```console
    yarn
  ```
  
  </li>
  <li>Para executar o frontend web (por padrão o projeto executa na porta 3000 do localhost):
  
  ```console
    yarn start
  ```
  
  </li>
</ol>
